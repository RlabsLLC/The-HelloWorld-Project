The Original Programming Language

Hello World is the programming language that first started when the first text on a screen was rendered: "hello world"

Why is it called HelloWorld?

Hello World was the first text that has ever been rendered on a screen in 1972. It has stuck around ever since.

What is HelloWorld?

Hello World is a programming language built in JavaScript that can run in the browser. HelloWorld unlocks the potential of JavaScript coding in a much simpler way.
