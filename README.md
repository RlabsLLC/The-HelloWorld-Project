## Overview

The HelloWorld Project is a new programming language focused on clarity, structure, and a distinct identity. It avoids the syntactic conventions of C-like or Python-like languages and instead introduces a declarative, expressive style built around quoted function signatures, grouped expressions, and a custom comment system.

The language is implemented in JavaScript and is fully open source under the HelloWorld Project Open Source License (HW-OSL-1.0).

---

## Language Features

### Function Definitions

Functions are defined using quoted signatures and parenthesis-based blocks.

```text
define "greet with person" (
    show {"Hello, " + person}
)
```
